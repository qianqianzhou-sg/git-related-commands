How can I change the author name / email of a commit?

 - Get Your Git Committer Name & Email Globally:
$ git config --global --get user.name
$ git config --global --get user.email


 - Change Your Git Committer Name & Email Globally:
$ git config --global user.name "John Doe"
$ git config --global user.email "john@doe.org"


 - Change Your Git Committer Name & Email per Repository:
$ git config user.name "John Doe"
$ git config user.email "john@doe.org"


 - Change the Git Author Information Just for the Next Commit:
$ git commit --author="John Doe <john@doe.org>"


Hint: For other situation and read more information read the post reference:
https://www.git-tower.com/learn/git/faq/change-author-name-email/
